# Business Intelligence Platform

Objetivo:
Criar uma plataforma SaaS multiempresa para transformar dados empresariais em insights.

Arquitetura:
Flutter -> FastAPI -> Business Intelligence Engine -> PostgreSQL

Engines:
- Auth
- Company
- Import
- Omie
- KPI
- Insight
- Alert
- Recommendation
- AI
- Notification

Princípios:
- Multi-tenant
- Escalável
- APIs REST
- Clean Architecture
